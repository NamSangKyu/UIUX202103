보강 1일차
    - js 입력과 출력
        input 태그, textarea, select, 버튼 태그로 입력 --> 데이터가 전부 문자열
        입력 받은 후에 데이터를 형변환
        숫자로 바꿀때 Number
        날짜는 Date
        
    - 변수 : 입력한 내용, 출력할 값, 계산한 결과 값,
             계산할려는 값, 태그, 함수, 객체......
        변수는 데이터든 객체든 함수든 저장할 할 수 있는 공간

    - 연산자 : 데이터(객체....)를 가지고서 계산 작업을 하기위한 코드
        사칙 연산자 : + - * / %
        복합대입 연산자 : += -= *= /= %= <-- 계산한 결과를 누적
            += --> 총합 구하는 목적?(백엔드 쪽에서 처리할때 주로 사용)
                --> 웹에서는 주로 태그를 조립하는 용도(문자열을 합치는 용도)
        증감 연산자 : ++ --   --->   반복문, 조건문에서 제어변수의 값을 바꾸는 용도
        관계(비교)연산자 : 조건식을 만드는 용도(제어문 제어하는 조건식을 작성하는 용도)
        논리 연산자 : 조건식을 2개이상 조합을 해야될때
